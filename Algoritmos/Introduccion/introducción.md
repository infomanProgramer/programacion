
# Introducción

En este apartado examinaremos el algoritmo de ordenación por ***Inserción***. Veremos por que este algoritmo ordena correctamente y analizaremos su *tiempo de ejecución*.

El análisis introduce una notación que describe como aumenta el tiempo de ejecución con el número de elementos a ordenar. Luego utilizaremos un método llamado ***Divide y Vencerás*** para desarrollar un algoritmo de ordenación llamado ordenación por ***Fusión*** y finalizaremos con el análisis de tiempo de ejecución de la ordenación por fusión
